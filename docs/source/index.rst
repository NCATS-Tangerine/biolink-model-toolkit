Welcome to Biolink Model Toolkit's documentation!
=================================================

BMT provides utility functions to query the Biolink Model for classes (subjects, objects, associations and mixins),
predicates, and both node and edge properties.

See the Installation section below for installing the toolkit and the Example Usage section for more examples.
